# Vignettes

Vignette source files (`.Rmd`) for PhenoMapR. These are built into articles by [pkgdown](https://pkgdown.r-lib.org/) and published at [brooksbenard.github.io/PhenoMapR](https://brooksbenard.github.io/PhenoMapR).

## Data requirements

Some vignettes require large data files that are not in the repo. Download and place in this directory:

- **Single-cell (GSE111672)**: `PAAD_GSE111672_seurat.rds` — [GEO GSE111672](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE111672)
- **Single-cell (CRA001160)**: `PAAD_CRA001160_seurat.rds` — [GSA CRA001160](https://ngdc.cncb.ac.cn/gsa/browse/CRA001160)
- **Bulk (GSE205154)**: `GSE205154.GPL20301.matrix.txt`, `GSE205154.info.txt` — [PRECOG](https://precog.stanford.edu/)
